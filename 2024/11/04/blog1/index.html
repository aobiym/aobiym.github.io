Ubuntu安装Nginx

1.下载Nginx

$ sudo apt update

$ sudo apt install nginx

2.调整防火墙

$ sudo ufw app list

没有的话要添加，使“Nginx HTTP”能通过防火墙

$ sudo ufw allow ‘Nginx HTTP’

$ sudo ufw status
